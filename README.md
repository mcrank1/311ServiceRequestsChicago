# **Chicago 311 Service Requests Analysis**

## **Project Overview**
This project is a simple python script that fetches **public Chicago 311 Service Request data** via the City of Chicago API, processes it using **Pandas**, and generates basic insights.

It demonstrates working with **real-world data**, **API integration**, and **data analysis**.


## **Features**
- **Fetches live Chicago 311 Service Requests** (limited to 500 records for simplicity)
- **Cleans and prepares data** using pandas
- Provides simple analytics:
   - **Total number of requests**
   - **Count of requests by type ('sr_type')**
   - **Daily trends of requests**
   - **5 Latest Requests**
